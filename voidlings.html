    ✅ Confirmed Voidlings

    These Voidlings have been confirmed through the free Steam demo, developer interviews (Jonathan Rancourt, Hatchery Games), and hands-on previews.

    Kwipeck 🐔 ✓ Confirmed (Demo)

    NeutralStarter

    Ability: Beak Minigun — long-range rapid-fire burst; Secondary: explosive bomb

    Playstyle: Nimble long-range DPS. Fast dodge, responsive shooter feel.

    Evolves to: Poison-sniper path (Toxic element, one-shot potential) via mutagen

    Variants: Mutated Kwipeck (Golden Egg) — pre-evolved, boosted stats, extra jump

    📝 Source: Pixelkin demo preview, Steam Demo

    Glick 🦎 ✓ Confirmed (Demo)

    Ice (Cryo path)Early Game

    Ability: Cone Blast — close-range shotgun spread; Special: leap attack

    Playstyle: Aggressive short-range brawler / shotgunner.

    Evolves to: Arctic Glick (Cryo path) — excels vs robotic enemies with freeze + shatter

    📝 Source: Pixelkin demo preview, Steam Demo

    Nimiod 🕊️ ✓ Confirmed (Dev Interview)

    UnknownMid Game

    Ability: Hover — sustained aerial mobility; melee-focused attacks

    Playstyle: Aerial melee. Requires special enemy design to accommodate hovering

    Obtain: Full game (not available in demo)

    📝 Source: RPGamer developer interview (April 2026)

    Gwigoon 🧬 ✓ Confirmed (Splicing)

    SplicedSpecial

    Ability: Gene-crafted — created via Splicing Station from multiple gene sources

    Obtain: Splicing Station (endgame system)

    📝 Confirmed spliced Voidling. See Splicing Guide for details.

    ☠️ Toxic Element Voidlings

    Toxic Voidlings specialize in poison, debuffs, and sustained damage over time. (Kerapin confirmed via dev interview)

    ToxicMid Game

    Ability: Burst Rifle — rapid-fire ranged attacks
